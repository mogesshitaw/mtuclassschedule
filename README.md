# Class Schedule Management System

## Project Overview
This project is a web-based solution designed to automate the complex task of university course scheduling. It replaces manual spreadsheet methods with an intelligent system that ensures conflict-free resource allocation.

## Tech Stack

Frontend: Next.js (React), Tailwind CSS

Backend: Node.js, Express

Database: PostgreSQL (with ACID compliance for data integrity)

State Management: React Hooks / Context API

## Key Features

Auto-Generation: Intelligent scheduling based on predefined constraints.

Conflict Notification: Real-time alerts for overlapping schedules or overbooked rooms.

Role-Based Access: Secure login for Department Heads and Instructors.

Interactive Interface: Dynamic UI for easy schedule adjustments and viewing.
